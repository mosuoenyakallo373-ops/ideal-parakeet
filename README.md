# Veltora

Online storefront for daily essentials, trip gear, accessories, and gadgets.

## Tech stack

- **Frontend:** React + Vite + Tailwind CSS
- **Database:** Supabase
- **Payments:** Paystack
- **Hosting:** Vercel

## Getting started

```bash
npm install
npm run dev
```

The dev server runs at `http://localhost:5173`.

## Build

```bash
npm run build
```

Output goes to the `dist` folder.
